# QoL: Unknown / Discord Rich Presence
**Quality of Life & Discord Rich Presence Mod for Casualties: Unknown**

<img width="458" height="188" alt="image" src="https://github.com/user-attachments/assets/fd932b1b-54d9-4570-a6b1-317cf7d19e38" />

- Toggle commands for most of the below, and new commands
- Discord embed
- Volume slider
- Fast-forwarding option for intro
- 1-tile water no longer pushes
- Console can now use up/down arrows to select different choices for autofill
- Crafting has a preview of items used, in containers
- Line of sight blends more (darker)
- Layer modifiers can be adjusted individually. Want only dried layers? You can do that 
- Luminosity and bloom can be adjusted
- Brain damage isn't as annoying
- **Shift + drag** to quickly move items between containers.  This also works to populate your material pouch or grav bag with anything that can fit
- **Shift + drag** to instantly move all liquids 
- Scroll bar for containers that are too long (i.e. material pouch)
- Inventory sort
- Optimizations that may or may not matter



# Installation: 
## Getting BepInEx:
1. Install BepInEx from [https://github.com/BepInEx/BepInEx/releases/download/v5.4.23.3/BepInEx_win_x64_5.4.23.3.zip](url)
2. Extract all files into your folder containing CasualtiesUnknown.exe. Replace all files when prompted
3. Run the game once, then close the game
